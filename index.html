Code Editing.
Redefined.

O Visual Studio Code é um editor de código-fonte leve, mas poderoso, que roda no seu desktop.

IntelliSense

Preenchimentos inteligentes baseados em tipos de variáveis, definições de funções e módulos importados.

Git Integrado

Trabalhe com Git e outros provedores de SCM diretamente do editor. Revise diffs e faça commits facilmente.

Extensões

Quer mais recursos? Instale extensões para adicionar novas linguagens, temas, depuradores e serviços adicionais.
